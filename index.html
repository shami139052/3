شبکه‌های اجتماعی:

آموزش کامل کدنویسی با GitHub

مراحل یادگیری:

  • نصب Git و تنظیم نام کاربری
  • ساخت مخزن با دستور git init
  • افزودن فایل‌ها با git add .
  • ثبت تغییرات با git commit -m "توضیح"
  • اتصال به GitHub با git remote add origin
  • ارسال کد با git push -u origin main
  • ساخت Branch و Pull Request برای همکاری تیمی
  • انتشار سایت با GitHub Pages

برای یادگیری بیشتر می‌تونی از دوره‌های رایگان مثل کدیاد یا سبزلرن استفاده کنی.
